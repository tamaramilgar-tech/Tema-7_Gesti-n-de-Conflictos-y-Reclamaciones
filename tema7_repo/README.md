# CYA · Tema 7 · Gestión de conflictos y reclamaciones

Repositorio web listo para subir a GitHub Pages o a cualquier hosting estático.

## Qué incluye

- Itinerario completo por fases.
- Guardado automático del progreso en `localStorage`.
- Fases bloqueadas hasta validación docente.
- Código de verificación diario con formato `ddmmaaaa`.
- Certificado final bloqueado hasta completar nombre y todas las fases.
- Mapa conceptual descargable para la fase previa.
- Prácticas de las fases 7 y 8 orientadas a vídeo o audio con al menos dos personajes.

## Estructura

- `index.html`
- `css/styles.css`
- `js/app.js`
- `js/storage.js`
- `js/verificacion.js`
- `js/fases.js`
- `assets/mapa_conceptual_tema7.svg`

## Uso

1. Abre `index.html` en navegador o súbelo a GitHub Pages.
2. El alumnado escribe su nombre y completa cada fase.
3. El alumnado sube la evidencia al EVAGD.
4. El docente facilita el código del día en formato `ddmmaaaa`.
5. La siguiente fase se desbloquea tras introducir el código correcto y aprobar el test.

## Personalización rápida

- Puedes editar contenidos y preguntas en `js/fases.js`.
- Puedes cambiar colores y diseño en `css/styles.css`.
- Si deseas otro sistema de códigos, edita `js/verificacion.js`.
